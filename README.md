# 中国象棋
| 学习编程练手
## 基本功能
1. 残局列表
2. 编辑棋局
3. 导出棋局：字符格式、图片格式
4. 悔棋
5. 重置棋局
6. 对弈记录
7. 记录回放
8. 棋局调整：上下左右対调，黑红对调
